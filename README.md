# Jason's Claude Code Skills

A collection of plugins and skills for [Claude Code](https://docs.anthropic.com/en/docs/claude-code).

## Plugins

| Plugin | Description |
|---|---|
| [academic-writing](./academic-writing) | Expert-level guidance for writing academic papers in engineering and computer science. Five skills covering the full paper lifecycle. |

## Installation

Add the marketplace, then install any plugin:

```
/plugin marketplace add jasonbian97/jason-cc-skills
/plugin install academic-writing@jason-cc-skills
```

## Standalone Skills

| Skill | Description |
|---|---|
| [commit-message](./commit-message) | Generate concise, informative git commit messages from staged changes. |
